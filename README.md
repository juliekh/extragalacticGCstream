### Code underlying the results of the paper "Evidence for the First Globular Cluster Stellar Stream beyond the Milky Way" (Holm et al., 2026)

Data_analysis: notebooks containing different parts of the data analysis, sorted by the end figure they produce (some have several). 

data_files: Additional files needed to run the data analysis scripts. If files are loaded in the scripts but are not in this folder, they are not mine but are available freely through online archives. This is true for the HST and CFHT data sets. 

simulation_files: files needed to run the simulation notebooks

simulations: notebooks illustrating the Xstream simulations, and creating the modelling figures in the paper
